Before You Deploy

You’ll need to install Spin - our open source developer tool - and have a Fermyon Cloud account (free) to deploy the application to.

Clone the Repo

```bash
git clone https://github.com/karthik2804/spin-zola.git && cd spin-zola
```

Deploy

```bash
spin cloud deploy
```
